Zexplorer (It may be offline now)

Zamunda torrent explorer web application. Optimized for mobile devices.
Since zamunda.net is quite PHP based, it provide already filled HTML templates. With the aid of jQuery and cheerio (jQuery core integrated for node.js), I extracting the needed data from the zamunda.net HTMLs to provide main services.
This application is alpha version yet. I am currently working on it.

Used technologies:
Server: Node.js (0.10.x) + express 4
Client: AngularJS + Angular Material + jQuery(full version).
